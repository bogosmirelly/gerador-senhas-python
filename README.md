# Gerador-Senhas-Python
Projeto desenvolvido em Python para geração de múltiplas senhas aleatórias e únicas, utilizando letras, números e caracteres especiais, com menu interativo no terminal.

# Objetivo

Aplicar conceitos básicos de programação em Python, como funções, laços de repetição, estruturas condicionais e uso de bibliotecas padrão, com foco em lógica de programação e fundamentos de segurança da informação.

# Funcionalidades
	•	Menu interativo no terminal
	•	Geração de múltiplas senhas aleatórias
	•	Senhas únicas (sem repetição)
	•	Uso de letras maiúsculas e minúsculas
	•	Uso de números
	•	Uso de caracteres especiais
	•	Limite de tamanho de senha (mínimo 4 e máximo 20 caracteres)
